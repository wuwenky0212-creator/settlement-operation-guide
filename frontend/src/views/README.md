# Views Documentation

## TransactionSummary.vue

交易汇总查询页面 - 用于查询和展示交易确认书列表。

### 功能特性

1. **查询条件表单**
   - 外部流水号（必填）
   - 交易状态、产品、交易对手等多个可选条件
   - 支持日期范围查询（交易日、起息日、到期日）
   - 支持货币、运营机构、清算方式等筛选

2. **交易列表表格**
   - 显示22个字段的交易信息
   - 斑马纹样式（白色/浅灰色交替）
   - 支持横向滚动
   - 点击行可查看交易详情

3. **分页控件**
   - 支持选择每页显示记录数（15/20/50/100）
   - 显示当前页码、总页数、总记录数
   - 首页、上一页、下一页、末页按钮

4. **导出功能**
   - 导出当前查询结果为Excel文件
   - 文件名自动包含日期

5. **错误处理**
   - 显示加载动画
   - 显示错误提示信息
   - 无结果时显示友好提示

### API集成

- **查询接口**: `GET /api/transactions`
- **导出接口**: `GET /api/export/transactions`

### 路由

- **路径**: `/transactions`
- **名称**: `transaction-summary`

### 使用方法

1. 在浏览器中访问 `http://localhost:5173/transactions`
2. 输入外部流水号（必填）
3. 选择其他可选查询条件
4. 点击"查询"按钮
5. 查看结果列表
6. 点击任意行查看交易详情
7. 点击"导出"按钮下载Excel文件

### 样式

- 主色调：红色 (#D32F2F)
- 表格斑马纹：白色和浅灰色交替
- 响应式设计：支持1366x768及以上分辨率
